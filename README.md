- Naive Bayes
- Support Vector Machine (SVM)
- LSTM (Deep Learning)

## Dataset
Sentiment140 dataset

## Tools & Libraries
- Python
- Scikit-learn
- TensorFlow / Keras
- Pandas, NumPy
